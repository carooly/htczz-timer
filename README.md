# 黄庭禅站桩-计时器

专为黄庭禅站桩设计的计时器Web应用程序，支持背景音乐、动态背景图片切换、多种主题和PWA功能，提供全屏沉浸式体验。

## 功能特性

### 核心功能
- **站桩计时器**: 专为黄庭禅站桩设计的正计时器，从00:00开始向上计时，精确到秒
- **开始/暂停/重置**: 完整的计时器控制功能
- **实时显示**: 大字体数字显示，清晰易读

### 用户界面
- **响应式设计**: 自动适应横屏和竖屏布局
- **控制面板**: 直观的按钮控制（开始/暂停、重置、布局切换、设置、帮助）
- **状态指示器**: 实时显示应用状态信息
- **全屏体验**: 专注于计时任务的沉浸式界面

### 背景系统
- **动态背景图片**: 支持多张背景图片自动切换
- **切换效果**: 淡入淡出、滑动、缩放三种切换动画
- **切换间隔**: 可自定义5-60秒的切换时间
- **随机播放**: 支持随机顺序切换背景图片

### 音频系统
- **背景音乐**: 支持多首背景音乐播放
- **音量控制**: 0-100%音量调节
- **播放模式**: 单曲循环、列表循环、随机播放
- **音乐开关**: 可随时启用或禁用背景音乐

### 主题与个性化
- **多种主题**: 默认、深色、浅色、蓝色、绿色五种主题
- **字体大小**: 可调节80%-150%的显示字体大小
- **背景遮罩**: 根据主题自动调整背景遮罩颜色

### 操作方式
- **触控操作**: 点击按钮控制，长按显示/隐藏控制面板
- **键盘快捷键**: 
  - 空格键: 开始/暂停
  - 回车键: 重置计时器
  - L键: 切换布局
  - S键: 打开设置
  - ESC键: 关闭设置或帮助
  - B键: 切换背景图片
  - M键: 切换音乐开关
  - H键: 打开帮助
- **双击操作**: 双击屏幕开始/暂停计时

### 离线功能
- **PWA支持**: 可安装为桌面应用
- **离线使用**: Service Worker缓存机制，支持离线运行
- **网络状态检测**: 自动检测并提示网络状态变化

## 使用说明

### 基本操作
1. 点击▶️按钮或按空格键开始计时
2. 点击⏸️按钮或按空格键暂停计时
3. 点击⏹️按钮或按回车键重置计时器
4. 点击↕️按钮或按L键切换横竖屏布局
5. 点击⚙️按钮或按S键打开设置面板
6. 点击❓按钮或按H键打开帮助面板

### 高级功能
- 长按屏幕1秒显示/隐藏控制面板
- 在设置面板中自定义背景、音乐、主题等选项
- 双击屏幕快速开始/暂停计时

## 开发与部署

### 本地运行
1. 将项目文件放在Web服务器目录下
2. 通过浏览器访问index.html
3. 确保浏览器支持Service Worker（HTTPS环境或localhost）

### 自定义配置
- 添加更多背景图片：在`img/`目录添加图片并在`script.js`的`loadBackgroundImages()`函数中注册
- 添加更多背景音乐：在`mp3/`目录添加音频文件并在`script.js`的`loadBackgroundMusic()`函数中注册
- 修改缓存版本：在`sw.js`中修改`CACHE_NAME`常量

## 操作方式

### 基本操作
- **开始/暂停计时**：点击开始/暂停按钮、双击屏幕任意位置或按空格键
- **重置计时器**：点击重置按钮或按回车键
- **显示/隐藏控制面板**：单击屏幕任意非按钮区域
- **切换屏幕布局**：点击布局按钮或按L键
- **打开设置面板**：点击设置按钮或按S键
- **打开帮助面板**：点击帮助按钮或按H键
- **关闭面板**：点击关闭按钮或按ESC键

### 背景图片设置
- **切换间隔**：可选择10秒、30秒、60秒或自定义间隔（5-300秒）
- **切换效果**：支持淡入淡出、滑动、缩放三种效果
- **随机播放**：可开启随机播放背景图片
- **图片选择**：在图片列表中可直接点击选择特定背景图片

### 键盘快捷键
- `空格键` - 开始/暂停计时
- `回车键` - 重置计时器
- `L键` - 切换屏幕布局（横屏/竖屏）
- `S键` - 打开设置面板
- `H键` - 打开帮助面板
- `B键` - 切换背景图片
- `M键` - 切换背景音乐开关
- `ESC键` - 关闭当前打开的面板
