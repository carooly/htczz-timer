# 背景图片使用说明

## 目录结构
- `/img/portrait/` - 存放竖屏设备背景图片
- `/img/landscape/` - 存放横屏设备背景图片

## 推荐图片尺寸
- 竖屏图片：1080x1920px
- 横屏图片：1920x1080px

## 使用方法
1. 准备符合推荐尺寸的背景图片
2. 将竖屏背景图片放入 `/img/portrait/` 目录
3. 将横屏背景图片放入 `/img/landscape/` 目录
4. 确保图片命名为 `bg1.jpg`, `bg2.jpg` 等格式
5. 刷新页面即可看到效果

## 注意事项
- 竖屏图片会在手机竖屏时显示
- 横屏图片会在手机横屏或PC端显示
- 图片会自动根据设备屏幕方向进行切换
